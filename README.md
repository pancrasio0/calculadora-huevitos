# calculadora-huevitos
Simulacion montecarlo en python para calcular probabilidad de beneficio al abrir huevitos en el ro

## COMO USAR

1. Editar item_data en main.py con los datos que recopiles manualmente.  Las probabilidades deben ser lo mas cercanas al 100% en total, ignora el kachua coupon.
2. Ejecutar main.py con python
3. Saca tus propias conclusiones :)
